DigiMuh Documentation
=====================

**DigiMuh** is a toolkit for ingesting, storing, and analysing dairy cow
environmental and physiological sensor data.  It consolidates
heterogeneous CSV exports from on-farm monitoring systems (smaXtec
rumen boluses, gouna respiration sensors, HerdePlus milking data)
into a single normalised SQLite database and provides a suite of
statistical analysis pipelines.
